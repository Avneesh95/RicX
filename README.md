# 🛒 RicX - MERN E-Commerce Platform

RicX is a full-stack e-commerce application built using the MERN stack, following industry-standard architecture and best development practices.

##  Features

*  Secure Authentication & Authorization using JWT and middleware
*  Razorpay Payment Gateway Integration
*  Automatic PDF Invoice Generation for every successful order
*  Product, Cart, and Order Management
*  Scalable Industry-Standard Folder Structure
*  RESTful APIs with proper MVC(Model-View-Controller) architecture
*  Protected Routes and Role-Based Access Control
*  Responsive and User-Friendly Design

##  Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* Razorpay
* JWT Authentication
* PDFKit (Invoice Generation)

##  Highlights

* Built with clean and maintainable code practices.
* Designed for scalability and real-world production environments.
* Complete e-commerce workflow from product browsing to payment and invoice generation.
